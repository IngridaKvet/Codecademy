Semantic sectioning tags

Tag Name Description
<header> header The header of a content section or the web page. The web page header often contains the website branding or logo.
<nav> navigation The navigation links of a section or the web page.
<footer> footer The footer of a content section or the web page. On a web page, it often contains secondary links, the copyright notice, privacy policy and -cookie policy links.
<main> main Specifies the main content of a section or the web page.
<aside> aside A secondary set of content that is not required to understand the main content.
<article> article An independent, self-contained block of content such as a blog post or product.
<section> section A standalone section of the document that is often used within the body and article elements.
<details> details A collapsed section of content that can be expanded if the user wishes to view it.
<summary> summary Specifies the summary or caption of a <details> element.
<h1> heading Headings on the web page. <h1> indicates the most important heading whereas <h6> indicates the least important.
